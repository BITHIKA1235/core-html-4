Ordered List of HTML Tags

  1. <html>: The root element of an HTML document. It wraps all the content of the page.
  2. <head>: Contains metadata and links to external resources like stylesheets. It is not visible on the web page.
  3. <title>: Sets the title of the web page that appears in the browser's title bar or tab.
  4. <body>: Contains the main content of the web page that is visible to users.
  5. <h1> to <h6>: Define HTML headings. <h1> is the largest, and <h6> is the smallest.
  6. <p>: Represents a paragraph of text.
  7. <a>: Defines a hyperlink that links to another page or resource. It requires an href attribute to specify the target URL.
  8. <img>: Embeds an image in the page. It requires a src attribute to specify the image source and an alt attribute for alternative text.
  9. <ul>: Creates an unordered list, typically displayed with bullet points.
  10. <ol>: Creates an ordered list, typically displayed with numbered items.
  11. <li>: Defines a list item in both unordered and ordered lists.
  12. <div>: A generic container for grouping and styling content. It does not have any inherent visual representation.
  13. <span>: A generic inline container for styling and grouping small parts of content. Unlike <div>, it does not break the flow of the content.
  14. <form>: Represents a form for collecting user input. It can contain various input elements like text fields, buttons, and checkboxes.
  15. <input>: Defines an input control. The type attribute specifies the type of input, such as text, password, or button.
